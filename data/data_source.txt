Dataset Source:
https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset

Description:
This Online Retail II data set contains all the transactions occurring for a UK-based and registered, non-store online retail between 01/12/2009 and 09/12/2011.The company mainly sells unique all-occasion gift-ware. Many customers of the company are wholesalers.

Usage in Project:
- Used Only 2010-2011 dataset not the complete dataset
- Imported using Python (Jupyter Notebook)
- Cleaned and transformed
- Loaded into SQL Server for analysis
- Connected to Power BI for dashboard creation

Dataset is not uploaded due to GitHub file size limitations (>25MB).
